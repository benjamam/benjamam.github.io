Projects

AgriTrak

My senior design project that I completed alongside three other classmates. AgriTrak is a mobile-first web app that allows the user to view relevant crop and farm metrics that allow the farmer to make informed decisions regarding his/her crops.

In addition, it was selected to be a part of the 2017 Sensors and Expo Conference in San Jose

AgriTrak was a full-stack project: it used hardware including Arduinos and sensors to measure and transmit the metrics, which were posted to a NoSQL database and displayed via the web app. The following technologies were used: HTML/CSS, PHP, jQuery, NoSQL db, Python, Arduino.

Characterizing and Detecting Meteors using Incoherent Scatter Radar (ISR)

I was able to spend a few weeks at Arecibo Observatory in Puerto Rico (the largest radio telescope in the world) doing research.

A group of classmates and I took raw data from experiments run at Arecibo and used this data to detect and characterize meteors in the ionosphere using MATLAB. Meteors cause interference when researchers attempt to collect data from deep in the solar system. By characterizing and filtering out meteor interference, researchers are able to collect much more accurate and significant data.
